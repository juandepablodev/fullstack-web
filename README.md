# Full-stack web app Gallery

## About the Project
This is a complete web application developed for my university course on Software and Information Engineering. The main idea was to create a platform where users can upload their photos, view them in a shared gallery, rate other people's work, edit or delete their own uploads, and manage their personal accounts. 

It was built as a learning project to combine front-end and back-end development in a real use case, following the exercises from our IISSI-2 laboratory sessions.

## Main Features
- **User Management:** Registration, login, and session control
- **Photo Upload:** Add new images with title, description, and category
- **Shared Gallery:** Responsive layout that shows all public photos
- **Rating System:** Users can rate each photo on a 1–5 scale
- **Edit & Delete:** Only the owner can modify or remove their uploads
- **Database Storage:** All users, photos, and ratings are saved in MariaDB

## Technologies Used
| Front-end | Back-end | Database & Tools |
|---|---|---|
| HTML5 & CSS3 | Python + Silence framework | MariaDB / MySQL |
| Bootstrap 5 | REST API with Axios/fetch | PHPMyAdmin (local) |
| JavaScript (ES6+) | Session management & auth | Git & GitHub |

## Course Context
This project follows the progressive sequence of the IISSI-2 laboratory course (2025/26), which covers:
- SQL scripts and db creating
- HTML & responsive CSS layouts
- JavaScript DOM manipulation and JSON handling
- AJAX/Fetch requests for GET and POST operations
- User authentication, session management, and API integration

It was designed to practice full-cycle web development while applying university best practices for code organization and version control.
